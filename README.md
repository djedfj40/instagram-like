# instagram-like

Bu proje Node.js(express) ile yazılmış instagram benzeri backend projesidir. Bu proje bir mobil uygulama projesine uygun şekilde geliştirilmiştir.

Kullanılan teknolojiler ve eklentiler:

 * Nodejs
 * Express
 * Javascript
 * MySQL
 * JWT
 * Crypto
 * express-fileupload
 * express-body-parser-error-handler
 * dotenv
 * validator

<hr>

Projede olan özellikler:

* Kullanıcı kaydı
* Kullanıcı girişi
* Kullanıcı şifre sıfırlama
* Kullanıcı şifre değiştirme
* Kullanıcının kendi bilgilerini görebilmesi
* Kullanıcı profil bilgileri değiştirme
* Kullanıcı profil fotoğrafı yükleme ve silme
* Kullanıcı hesap silme
* Kullanıcının başka profillere bakabilmesi
* Kullanıcının diğer kullanıcıları ismiyle arayıp bulabilmesi
* Kullanıcının diğer kullanıcıları rapor edebilmesi
* Kullanıcının gönderi yükleyebilmesi
* Kullanıcının gönderi silebilmesi
* Kullanıcının diğer kişilerin yüklediği gönderileri görebilmesi ve beğenip yorum yapabilmesi
* Kullanıcının yaptığı yorumu silebilmesi
* Kullanıcının diğer gönderileri kaç kişinin beğendiğini ve yorum yaptığını görebilmesi
* Kullanıcının diğer kişilerin gönderideki yorumlarını görebilmesi
